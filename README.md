# VibeGuide ✨

> **一款像素风 RPG 冒险游戏，带你从零开始学会 Vibe Coding。**

在数字大陆上，AI 精灵苏醒了——它们能听懂人话、帮你写代码。

你不需要会编程，只需要**说出你想要什么**。

这就是 **Vibe Coding**。而 VibeGuide，是你的冒险起点。

---

##  世界观

*在遥远的数字大陆上，人们用"代码语言"构建网站、应用和游戏——但这门古老的语言只有少数人能掌握。*

*直到有一天，AI 精灵苏醒了。它们能理解人类的自然语言，将想法直接转化为代码，让每一个人都拥有了「创造」的力量。*

*现在，数字大陆需要新的建造者。*

*勇者，你准备好了吗？*

##  这游戏怎么玩？

**VibeGuide 不是模拟器——你做的每件事都是真的。**

安装 IDE、创建项目、启动预览、部署上线——全是**真实操作**，NPC 一步步带你做。

### 冒险流程

```
️ 起源村        →  创建角色，选择你的职业
⚒️ 工具锻造谷    →  锻造你的武器（安装 AI 编程 IDE）
️ 觉醒神殿      →  觉醒与 AI 对话的能力
 创造平原      →  用 AI 造出一个真正的网站
 传送灯塔      →  把你的网站传送到全世界
```

##  Vibe Coding 三原则

贯穿游戏的核心理念：

1. **描述你想要什么，不需要先知道怎么做**
2. **遇到问题把报错贴给 AI，让 AI 修**
3. **不断迭代，一次只提一个需求**

---

## ✨ 特色

-  **全程序化 8-bit 音乐** — 没有音频文件，BGM 和音效全部用代码实时合成
-  **像素方块粒子特效** — 烟花、爆炸、星芒、升级光柱、彩纸飘落
-  **智能检测** — 自动检测你的 IDE、文件、端口，不用手动汇报
-  **NPC 对话引导** — 打字机效果，一步步带你
-  **一键复制 AI 咒语** — 把 Prompt 直接粘贴给 AI 就行
-  **自动存档** — 关了游戏下次接着玩
- ⚔️ **装备 & 徽章** — 完成挑战就有奖励

##  下载 & 启动

### 方式一：直接玩

前往 [Releases](../../releases) 下载最新的 `.exe`，双击启动，开始冒险。

### 方式二：从源码运行

需要 [Node.js](https://nodejs.org/) 18+。

```bash
git clone https://github.com/你的用户名/VibeGuideGame.git
cd VibeGuideGame
npm install
npm run dev
```

### 打包成可执行文件

```bash
npm run build:electron
```
