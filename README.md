## 💡 Vibe Coding 三原则

贯穿整个冒险的核心理念：

| # | 原则 | 翻译成人话 |
|:-:|------|-----------|
| 1 | **描述你想要什么** | 不需要先知道怎么做，说出来就行 |
| 2 | **报错贴给 AI** | 遇到 bug？截图 + 复制错误丢给 AI，它来修 |
| 3 | **一步一步来** | 别一次提 10 个需求，一次一个，迭代前进 |

## ✨ 一些内容

- 🎵 **全程序化 8-bit 音乐** — 零音频文件，BGM 和音效全部用 Web Audio API 实时合成
- 💥 **像素方块粒子特效** — 烟花、爆炸、星芒、升级光柱、彩纸飘落，通关时的烟花能放一整天
- 🔍 **智能检测** — 自动检测你的 IDE、文件、端口，游戏知道你做到哪了
- 💬 **NPC 打字机对话** — 一个字一个字蹦出来，很有氛围
- 📋 **一键复制 AI 咒语** — 游戏给你准备好 Prompt，复制粘贴给 AI 就行
- 💾 **自动存档** — 关了游戏下次接着冒险，不会丢进度
- 🏆 **装备 & 徽章** — 每关通关都有奖励，集齐装备成就感拉满

## 🚀 开始冒险

### 方式一：直接玩（推荐）

前往 [Releases](../../releases) 下载最新的 `.exe`，双击启动，开始冒险。

### 方式二：从源码运行

需要 [Node.js](https://nodejs.org/) 18+。

```bash
git clone https://github.com/Damue01/VibeGuideGame.git
cd VibeGuideGame
npm install
npm run dev
```

### 打包成可执行文件

```bash
npm run build:electron
```
